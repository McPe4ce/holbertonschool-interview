# Holberton School – Interview Preparation (Python)

A collection of **technical interview practice problems** implemented in Python.

Each directory contains a focused challenge (algorithmic/logic-based), typically matching Holberton School interview-style tasks.

## Repository structure

- `lockboxes/` – Classic “Lockboxes” reachability problem
- `minimum_operations/` – Operations-counting / greedy-style problem
- `pascal_triangle/` – Pascal’s Triangle generation and related tasks

## Getting started

### Prerequisites

- Python 3.x
- A Unix-like environment (Linux/macOS/WSL recommended)

### Clone

```bash
git clone https://github.com/McPe4ce/holbertonschool-interview.git
cd holbertonschool-interview
```

## How to run

Enter a problem directory and run the relevant script(s). Example:

```bash
cd lockboxes
python3 0-lockboxes.py
```

## Notes

- This repository is intended for practice and mastery; solutions aim for clarity and correctness.
- Some tasks may include specific input/output formats based on the original prompt.
